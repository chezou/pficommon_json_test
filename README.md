Sample code of JSON serialization and deserialization with pficommon
==========================
2012 chezou

REQUIREMENT
--------------------

* [pficommon](https://github.com/pfi/pficommon)
* python (for waf)

HOW TO BUILD
--------------------
    $ ./waf configure
    $ ./waf

HOW TO RUN
--------------------
    $ ./build/src/json-test
